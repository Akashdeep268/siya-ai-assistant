# Siya AI 🤖

Siya AI is a personal AI companion chatbot developed using Python and Large Language Models (LLMs).

The project focuses on creating a more natural and personalized conversational experience through memory, personality adaptation, mood detection, and learning from user interactions.

## Features

- AI-powered conversations
- Long-term memory system
- Personality customization
- Mood detection
- User profile learning
- Telegram bot integration
- Vector memory search
- Multi-language conversational support
- Human-like response delays

## Technologies Used

- Python
- OpenAI / LLM APIs
- Telegram Bot API
- JSON Memory Storage
- Sentence Transformers
- FAISS Vector Search

## Project Structure

- bot.py – Main chatbot logic
- memory.json – User memory storage
- personality.json – Personality configuration
- learning.txt – Learned conversations
- requirements.txt – Project dependencies
